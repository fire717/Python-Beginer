## Jupyer Notebook

### 常用

* [本地打开服务器的jupyter](https://amber-md.github.io/pytraj/latest/tutorials/remote_jupyter_notebook)
  
* 切换其他conda环境
  1. 在cmd中切换到想要的环境，比如说pth
  2. 在pth环境中中安装好ipykernel: conda installipykernel
  3. 执行：python -m ipykernel install --name pth
