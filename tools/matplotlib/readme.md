## 常用

* 隐藏坐标轴
```
plt.axis('off')
```

* 3d图调整图像大小
```
fig,_ = plt.subplots(figsize = (40,40))
ax = plt.subplot(projection='3d')
```

* 保存plt图片
```
plt.savefig('save.jpg')
```
